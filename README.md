# Email-classification
Goal of this project was to classify a sentence (emails) to particular classes. The model consist of the GRU, Embeddings and Dropout layers. The f1-score on test set was 0,95.

*numpy
*matplotlib
*pandas
*nltk
*tensorflow-text
*scikit-learn
